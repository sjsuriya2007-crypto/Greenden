# 🌿 Greenden - Responsive Plant Store Website

Greenden is a modern and responsive plant store website built using **HTML5** and **Tailwind CSS**. The project features a clean, user-friendly interface that allows users to explore a variety of plants while demonstrating responsive web design and modern frontend development practices.

## 🚀 Features

* 🌱 Responsive Navigation Bar
* 🏡 Attractive Hero Section
* 🌿 Plant Product Gallery
* 🔍 Search Bar
* ⭐ Customer Reviews Section
* 📞 Contact Section
* 📱 Fully Responsive Design
* 🎨 Modern UI with Tailwind CSS

## 🛠️ Technologies Used

* HTML5
* Tailwind CSS
* Responsive Web Design
